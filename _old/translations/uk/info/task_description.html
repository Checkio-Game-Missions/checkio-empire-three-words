Давайте навчимо наших роботів відрізняти слова від чисел.

Заданий рядок зі словами і числами, розділеними пробілами (один пробіл між словами та / або числами). Слова складаються тільки з букв. Вам потрібно перевірити чи є у вихідному рядку три слова підряд. Наприклад, в рядку "start 5 one two three 7 end" є три слова підряд.

Зауваження: Ви легко вирішите це завдання використовуючи дані функції: {% if "python 2" in interpreter|lower %} str.split, str.isalpha и str.isdigit. {% else %} str.split, str.isalpha и str.isdigit. {% endif %}

Вхідні дані: Рядок зі словами (str).

Вихідні дані: Відповідь як логічне значення (bool), True або False.

Приклади: 

checkio("Hello World hello") == True
checkio("He is 123 man") == False
checkio("1 2 3 4") == False
checkio("bla bla bla bla") == True
checkio("Hi") == False

Як це використовується: Це завдання підкаже вам як працювати з рядками, і покаже деякі корисні функції.

Передумови: Вихідний рядок містить тільки слова і / або числа. Змішаних слів немає (тобто слів з перемішаними цифрами і буквами).
